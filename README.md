# 前言

大家好，今天我要分享的是一个关于新能源汽车个性化推荐系统的毕业设计项目。该项目基于Java语言和MySQL数据库开发，采用Spring Boot框架，前端技术包括JS、Vue和CSS3。在这个项目中，我致力于为大家提供一种便捷、高效的新能源汽车推荐方案。

# 内容介绍

新能源汽车个性化推荐系统主要实现了以下功能：

1. 用户注册、登录以及个人信息管理；
2. 新能源汽车各类数据展示，包括品牌、车型、续航里程等；
3. 用户可根据个人喜好和需求进行筛选，系统将根据用户行为和车辆数据实现智能推荐；
4. 系统后台管理，包括用户管理、车辆数据管理以及推荐算法的优化。

通过这个项目，我希望能够帮助大家更好地了解新能源汽车，为大家的购车过程提供指导。

# 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16

# 核心代码

private CarRepository carRepository;
